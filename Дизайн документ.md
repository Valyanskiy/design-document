# Название проекта
Sky Jump
# Короткое описание
Простой low-poly 3D платформер, в котором игрок перемещается по островам, парящим в небе, прыгая по платформам, избегая препятствий и собирая кристаллы.
# Платформа и целевые устройства
В качестве целевой платформы выбрана PC (macOS).
Целевые характеристики устройства:
- CPU: Apple M1
- RAM: 8 GB
- GPU: Apple M1
- Разрешение: 1920×1080
# Жанр и сеттинг
Жанр: **3D платформер**
Сеттинг: Фантастический мир из парящих островов в небе. Игрок перемещается между платформами, скалами и мостами, собирая кристаллы и продвигаясь дальше по уровню.
# Игровой цикл
Основной игровой цикл состоит из следующих действий:
1. Игрок начинает уровень на стартовой платформе.
2. Исследует уровень и прыгает между платформами.
3. Собирает кристаллы (collectibles).
4. Избегает препятствий (шипы, падающие платформы).
5. Активирует чекпоинты.
6. Доходит до финальной точки уровня.
7. Получает результат (количество собранных кристаллов).
8. Переходит на следующий уровень.

# Требования к материалам  - PBR палитра, подход к атласам/trim sheets, утверждённые цветовые схемы
Используется **PBR-подход**.
Основные параметры материалов:
- Base Color (Albedo)
- Normal Map
- Metallic
- Smoothness
Стиль:
- Low-poly
- Простые цвета
- Минимум сложных текстур
Подход:
- Использование **texture atlases** для пропсов
- Общая цветовая палитра:
    - голубой (небо)
    - зелёный (острова)
    - серый (камни)
    - яркий голубой (кристаллы)
